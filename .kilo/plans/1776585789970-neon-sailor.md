# Plan: Minimal Luxury Sensual Spa Theme Redesign

## Overview
Redesign the existing spa website to follow a minimal, luxury, sensual aesthetic. The goal is to make the design feel premium, soft, and suggestive—not explicit or cluttered.

## Current State Analysis

### What's Already in Place
- Dark theme with `#0a0a0a` background (similar to proposed `#0F0F10`)
- Gold accent `#c9a227` (similar to proposed `#C6A96B`)
- Serif font for headings via `font-heading` class (currently Cormorant Garamond, changes to Playfair Display)
- Sans-serif for body (Inter)
- Full-height hero section already exists
- Therapist cards with hover effects already exist
- Services section already exists

### Key Differences to Address
| Aspect | Current | Proposed |
|--------|---------|----------|
| Heading font | Cormorant Garamond | Playfair Display |
| Background | `#0a0a0a` | `#0F0F10` |
| Secondary bg | `#121212` | `#1A1A1D` |
| Accent gold | `#c9a227` | `#C6A96B` |
| Therapist cards | Column-based grid | Elegant hover overlay |
| Services | Grid with cards | Simple 3-column text |

## Implementation Plan

### Phase 1: Global Theme Updates

#### 1.1 Update CSS Variables (globals.css)
```css
/* Replace current colors */
--background: #0F0F10;      /* was #0a0a0a */
--background-alt: #1A1A1D; /* was #121212 */
--primary: #C6A96B;         /* was #c9a227 */
--primary-hover: #D4B87A;
--text-primary: #F5EFE6;
--text-secondary: #A8A29E;
--text-muted: #737373;

/* Update fonts */
--font-heading: 'Playfair Display', serif;
--font-body: 'Inter', sans-serif;
```

#### 1.2 Update Font Imports
- Switch from Cormorant Garamond to Playfair Display
- Keep Inter for body text

### Phase 2: Page-by-Page Updates

#### 2.1 Home Page (page.tsx)
- **Hero**: Keep full-screen, update gradient overlay (`bg-black/60` instead of current custom)
- **Hero text**: Shorten to "Relax. Indulge. Escape." style tagline
- **Therapists/Models section**: Convert to elegant hover overlay cards
- **Services/Our Packages**: Simplify to 3-column clean layout
- **Reviews**: Keep horizontal scroll, make minimal
- **CTA**: Keep minimal form

#### 2.2 Services Page (services/page.tsx)
- Simplify to clean 3-column layout without complex cards
- Keep essential info: name, price, duration, description
- Remove redundant UI elements

#### 2.3 Therapists Page (therapists/page.tsx)
- Convert to elegant card grid with hover overlay
- Soft lighting images only
- Add hover zoom + fade overlay

#### 2.4 Booking Page (booking/page.tsx)
- Keep minimal form styling
- Update input backgrounds to `#1A1A1D`
- Clean border styling

### Phase 3: Cleanup Tasks

#### 3.1 Remove/Reduce
- Bright/red/neon colors (replace with gold muted tones)
- Cluttered sections
- Long text paragraphs (keep short + premium)
- Overly complex card designs

#### 3.2 Ensure Mobile Responsiveness
- All components must be responsive
- Touch-friendly spacing

### Phase 4: Visual Consistency

#### 4.1 Apply Globally
- Border radius: `rounded-2xl`
- Spacing: `py-16 px-6`
- Transition: `all 0.3s ease`
- Shadows: dark subtle shadows
